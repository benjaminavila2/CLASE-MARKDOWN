# CLASE-MARKDOWN github 

es una plataforma en la nube diseñada para desarrolladores, permite almacenar, gestionar y compartir codigo fuente utilizado en sistema de control de versiones Git
funciona como red social para programadores, facilita la colaboración, el seguimiento de cambios y el desarrollo de proyectos de software, sean publicos (codigos abiertos) o privados


MARKDOWN: lenguaje de marcado ligero
es una forma de escribir texto con formato (negritas, titulos, listas, incluir enlaces o imagenes)


SALTOS DE LINEAS: separar dos parrafos: enter dos veces
